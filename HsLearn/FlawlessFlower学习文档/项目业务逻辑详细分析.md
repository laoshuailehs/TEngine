## 1. 项目业务概述

### 1.1 这是什么项目？

这是一个**游戏机台自助服务终端**项目，类似于赌场/游戏厅中的现金兑换机（Kiosk）。它运行在 Android 机台上，连接到云服务器，提供以下核心业务：

```
                    ┌──────────────────────────────┐
                    │       MoneyBox 自助终端       │
                    │                              │
  玩家 ──────────→  │  ① 投纸币 → 充值到虚拟账户    │──→ 云端服务器
                    │  ② 扫二维码 → 现金兑换(出钞)  │
  店员 ──────────→  │  ③ 管理后台 → 查账/设置/打印  │
                    │  ④ 创建玩家账户 → 打印凭条    │
                    └──────────────────────────────┘
```

### 1.2 核心业务流程全景图

```
┌─────────────────────────────────────────────────────────────────────────┐
│                            系统启动                                       │
│  开机 → SBox硬件初始化 → HTTP登录 → MQTT连接 → 获取配置 → 主界面         │
└─────────────────────────────────────────────────────────────────────────┘
                                    │
                    ┌───────────────┼───────────────┐
                    ▼               ▼               ▼
            ┌─────────────┐ ┌─────────────┐ ┌─────────────┐
            │  投币充值    │ │  现金兑换    │ │  管理后台    │
            │  (纸币入金)  │ │  (扫码出钞)  │ │  (店员操作)  │
            └──────┬──────┘ └──────┬──────┘ └──────┬──────┘
                   │               │               │
                   ▼               ▼               ▼
            ┌─────────────────────────────────────────────┐
            │              打印机出票                       │
            │  充值凭条 / 兑换凭条 / 账目报表 / 玩家卡      │
            └─────────────────────────────────────────────┘
```

### 2.2 MainGameView 启动流程 (Start方法)

```csharp
void Start()
{
    // ① 启动局域网联机 (作为主机)
    NetMgr.Instance.SetNetAutoConnect(true);  // 端口7789 WebSocket服务 + 18000 UDP广播

    // ② 缓存 MenuObj 子对象到字典 (用于后续菜单权限控制)
    foreach (Transform item in Obj.transform)
        MainMenuObjs.Add(item.name, item.gameObject);

    // ③ 注册所有按钮点击事件 → 统一入口 OnBtnClick(btnName)
    Button[] buttons = transform.GetComponentsInChildren<Button>();
    foreach (var item in buttons)
        item.onClick.AddListener(() => OnBtnClick(item.name));

    // ④ 初始化图片轮播
    PhotoCarouselRawImage = transform.Find("PhotoCarousel").GetComponent<RawImage>();

    // ⑤ 语言标志和配置读取
    CommentConfigRead();  // 从本地文件读取配置(语言/纸币器/打印机设置)

    // ⑥ 初始化二维码输入框 (用于扫码枪)
    QRCodeInputField = transform.Find("QRCodeInputField").GetComponent<InputField>();
    QRCodeInputField.ActivateInputField();  // 自动获取焦点

    // ⑦ 显示机台编号和版本号
    SetIDText = GameObject.Find("SetIDText").GetComponent<Text>();
    ProjectVersionText = transform.Find("ProjectVersionText").GetComponent<Text>();
    ShowProjectVersion();  // 显示 "Ver:主版本.子版本.修订.资源版本"

    // ⑧ 初始化网络错误提示文本引用
    NetDisconnetTipText = GameObject.Find("NetDisconnetTipText").GetComponent<Text>();
    PublicFailTipText = GameObject.Find("PublicFailTipText").GetComponent<Text>();

    // ⑨ Android 平台特殊处理
    if (Application.platform == RuntimePlatform.Android)
        WifiShow();  // 显示WiFi信号强度

    // ⑩ 注册硬件按键 (SET键 → 打开管理后台)
    if (!Application.isEditor)
        SBoxSandboxListener.Instance.AddButtonClick(SBOX_SWITCH.SWITCH_SET, OpenBGMgr);
}
```

### 2.3 Update 循环逻辑

```csharp
void Update()
{
    // ──── 鼠标点击处理 ────
    // 通过射线检测判断点击的UI元素，存入 ClickedObj
    if (Input.GetMouseButtonDown(0) && IsEnbleClick)
    {
        IsEnbleClick = false;
        StartCoroutine(ClickFunc());  // 下一帧恢复 IsEnbleClick = true
    }

    // ──── 二维码扫码输入检测 ────
    // 当扫码枪输入内容到 InputField 时自动触发处理
    if (QRCodeInputField.text.Length > 0 && IsEnableQRCode)
    {
        IsEnableQRCode = false;
        StartCoroutine(IsContainHead());  // 检测是否包含 qr_code: 或 bank: 头
    }

    // ──── 保持二维码输入框始终获取焦点 ────
    if (!IsQRCodeHead && !QRCodeInputField.isFocused)
        QRCodeInputField.ActivateInputField();

    // ──── 编辑器调试快捷键 ────
    if (Application.isEditor)
    {
        F1 → 模拟投纸币100元
        F2 → 模拟纸币存入钱箱 (连续4次)
        F3 → 打开管理后台
        F4 → 显示钱箱预警提示
    }
}
```

### 4.3 纸币验证详情 (OnBillInAsync)

```csharp
IEnumerator OnBillInAsync(int redit)  // redit = 纸币面额
{
    // ═══ 步骤1: 前置检查 ═══
    if (recharging) return;                    // 正在充值中，忽略
    recharging = true;

    // 检查网络连接 (非编辑器环境)
    if (!Application.isEditor)
    {
        if (!IsCanInputMoney || 无网络)
        {
            SBoxSandbox.BillReject();          // 拒收纸币
            recharging = false;
            return;
        }
    }

    // 锁定UI点击
    MainGameView.IsEnbleClick = false;

    // ═══ 步骤2: 打印机检查 ═══
    if (string.IsNullOrEmpty(Net.publicMesg.prcode))  // 本地无预订单
    {
        if (!PrinterScript.CheckPrinterStatus())     // 打印机故障
        {
            SBoxSandbox.BillReject();                // 拒收
            MainGameView.IsEnbleClick = true;
            recharging = false;
            return;
        }

        // ═══ 步骤3: 请求新充值码 ═══
        Task task = Net.PublicWithRe(Net.GetNewPrcode, "{}");
        yield return new WaitUntil(() => 收到响应);

        if (Net.retData.Code != 0)  // 请求失败
        {
            SBoxSandbox.BillReject();
            recharging = false;
            yield break;
        }

        // 保存订单信息
        curBillStackedNum = 1;
        Net.publicMesg.prcode = Net.retData.Data.prcode;       // 充值码
        LocalOrderState.localOrderInfo.precode = prcode;
        LocalOrderState.localOrderInfo.time = 当前时间;

        // 更新充值/兑换限额
        MaxRecharge = Net.retData.Data.rangeInfo.maxRecharge;
        MinRecharge = Net.retData.Data.rangeInfo.minRecharge;
    }

    // ═══ 步骤4: 金额上限检查 ═══
    if (累计金额 > MaxRecharge)
    {
        SBoxSandbox.BillReject();
        EnableErrorMsg("超过最大充值限额");
        recharging = false;
        yield break;
    }

    // ═══ 步骤5: 逐张纸币验证 ═══
    CheckPerCashIn checkData = new CheckPerCashIn
    {
        platTag = Net.publicMesg.platTag,
        amount = redit,                    // 当前纸币面额
        prcode = Net.publicMesg.prcode,    // 充值码
        playerId = Net.publicMesg.playerId,
        // ...
    };
    Task taskB = Net.PublicWithRe(Net.HD_CheckPerCashIn, 加密JSON);
    yield return new WaitUntil(() => 收到响应);

    if (Net.retData.Code != 0)  // 验证失败
    {
        SBoxSandbox.BillReject();
        EnableErrorMsg(错误信息);
        recharging = false;
        yield break;
    }

    // ═══ 步骤6: 验证通过 ═══
    EventCenter.Instance.EventTrigger(EventHandle.BILL_ACCEPT, redit);
    // 等待纸币实际存入钱箱 → OnBillStacked() 处理
}
```

## 7. 本地订单持久化与补发机制

### 7.1 为什么需要本地订单？

在网络不稳定或打印失败时，交易可能已完成但凭条未打印。本地订单机制确保：

1. **断电保护**: 订单先保存到本地文件，再执行网络请求
2. **断网恢复**: 联网后自动补发未完成的订单
3. **打印重试**: 打印失败后可以自动或手动重新打印

### 7.2 本地订单数据结构

### 7.5 打印重试机制

```csharp
IEnumerator WaitPrintEnd(int waitSec)
{
    yield return new WaitForSeconds(waitSec);  // 等待打印完成

    // 最多重试18次 (每秒检查一次打印机状态)
    for (int i = 0; i < 9; i++)
    {
        if (CheckPrinterStatus()) break;
        yield return new WaitForSeconds(1);
    }

    if (打印机正常)
        DestroyLocalOrder(fileName);  // 删除本地订单
    else
    {
        ReprintCount++;
        if (ReprintCount < 2)
        {
            // 再重试5次
            for (int i = 0; i < 5; i++)
            {
                yield return new WaitForSeconds(1);
                if (CheckPrinterStatus())
                {
                    ClearPrintBufferFunc();  // 清打印缓冲区
                    yield return new WaitForSeconds(10);
                    return;
                }
            }
        }
        EnableErrorMsg("打印机故障, 请联系工作人员");
    }
}
```

```csharp
{

    StatusReport status = printer.GetStatus(StatusTypes.OfflineStatus);
    int sboxState = SBoxSandbox.PrinterState();

    if (!status.IsPaperLevelOkay)  → "打印纸不足"
```
