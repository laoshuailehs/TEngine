## 二、SBox 命令包结构

```csharp
// SBoxPacket - 与算法卡通信的数据包
class SBoxPacket
{
    int cmd;      // 命令码
    int source;   // 源地址 (1=主机)
    int target;   // 目标地址 (2=算法卡)
    int size;     // 数据大小
    int[] data;   // 数据载荷
}

// 发送流程
SBoxPacket packet = new SBoxPacket(cmd: 20040, source: 1, target: 2, size: 2);
packet.data[0] = 0;
packet.data[1] = 0;
SBoxIOEvent.AddListener(packet.cmd, BattleGetStateR);  // 注册回调
SBoxIOStream.Write(packet);                             // 发送
```

### 命令码速查表

| 命令码 | API 方法 | 功能 | 触发时机 |
|--------|---------|------|---------|
| 20040 | `BattleGetState()` | 读取游戏状态（总押注、幸运玩家等） | 循环轮询(0.08~0.25s) |
| 20041 | `BattleGameNumber()` | 读取当前轮数和局数 | 启动时/每局开始 |
| 20042 | `BattleGetCompletedGame()` | 读取已开奖的100局结果 | 初始化时 |
| 20043 | `BattleResetGame()` | 重置游戏（开始押注） | Prepare→Bet 时 |
| 20044 | `BattleNewRound()` | 开启新一轮（打印路单） | 每100局开始 |
| 20045 | `BattleEndRound()` | 中止当前轮 | 错误时/主动结束 |
| 20046 | `BattleRequestResult()` | 请求开奖结果 | (预留) |
| 20047 | `BattleLeadStart()` | 开牌（请求本局结果） | 倒计时结束后 |
| 20048 | `BattleLeadStop()` | 开牌结束通知 | 结果动画播放后 |
| 20049 | `BattLuckShow()` | 显示小游戏（卡牌选择） | 中幸运座位后 |
| 20050 | `BattLuckPrize()` | 开小游戏结果 | 玩家选牌后 |
| 20051 | `BattPrinterOpenBox()` | 打印机开箱 | 检查阶段/手动 |
| 20052 | `BattlePlayerInState()` | 上报玩家状态 | 状态变化时/300ms定时 |
| 20053 | `BattlePlayerOutStateListener()` | 监听玩家输出状态 | 启动时注册一次 |
| 20055 | `GetPlayerTotalBet()` | 读取所有玩家总押注 | 开奖后 |

## 三、硬件按键 (SBOX_SWITCH)

```csharp
// SandboxController.cs 中使用的按键常量
class SBOX_SWITCH
{
    // 押注按键 (5门)
    SWITCH_A    // 黑桃 Spade
    SWITCH_B    // 红心 Heart
    SWITCH_C    // 梅花 Club
    SWITCH_D    // 方块 Diamond
    SWITCH_E    // 皇冠 Crown

    // 功能按键
    SWITCH_ENTER        // 确认键
    SWITCH_ESC          // 退币键 (也是中奖后闪烁灯)
    SWITCH_PAYOUT       // 退币键
    SWITCH_UP           // 切换倍率(上)
    SWITCH_DOWN         // (下)
    SWITCH_LEFT         // 切换帮助面板
    SWITCH_RIGHT        // (右)
    SWITCH_ROOT_SET     // 根设置键 (长按触发错误)
    SWITCH_RULE         // 打印键
    SWITCH_SET          // 设置键 (长按进入设置)
    SWITCH_KEYBOARD_LITTLE_GAME  // 小游戏/账单键

    // IO面板键盘
    SWITCH_KEYBOARD_UP
    SWITCH_KEYBOARD_DOWN
    SWITCH_KEYBOARD_LEFT
    SWITCH_KEYBOARD_RIGHT
    SWITCH_KEYBOARD_CONFIRM   // 确认
    SWITCH_KEYBOARD_CANCLE    // 取消/手动打印
}
```

### 按键事件类型

```csharp
// SandboxController.cs - AddHardwareBtnEvent()

// 1. 按下事件 (AddButtonDown)
//    手指按下时触发一次
SBoxSandboxListener.Instance.AddButtonDown(SWITCH_A, () => {
    PlayerMgr.Instance.localPlayer.LocalPlayerChangeInState(KEY_A, true);
});

// 2. 松开事件 (AddButtonUp)
//    手指松开时触发一次
SBoxSandboxListener.Instance.AddButtonUp(SWITCH_A, () => {
    PlayerMgr.Instance.localPlayer.LocalPlayerChangeInState(KEY_A, false);
});

// 3. 点击事件 (AddButtonClick)
//    按下+松开 = 一次完整点击
//    用于功能键（设置、打印、确认等）

// 4. 长按事件 (AddButtonLongPress)
//    按住超过阈值时间触发
//    用于：设置（长按进入）、取消（长按触发错误暂停）
```

### 按键 → 控制器事件流

```
硬件按键按下
    → SBoxSandboxListener 检测
        → 触发对应回调
            → 回调中调用 EventCenter.EventTrigger(EventHandle.HARDWARE_KEY_CLICK, ...)
                → Controller.OnHardwareKeyClick(ulong SBoxSwitch)
                    → 根据当前 FSM 状态分发到具体处理函数
```

### Controller.OnHardwareKeyClick 分发逻辑

```csharp
// Controller.cs
private void OnHardwareKeyClick(ulong SBoxSwicth)
{
    if (FSMSystem.Instance.CurrentStateID == StateID.Setting) return;

    switch (SBoxSwicth)
    {
        case SWITCH_RULE:                   view.OnPrintBtnClick();    break;
        case SWITCH_KEYBOARD_LITTLE_GAME:
        case SWITCH_SET:                    view.OnBillBtnClick();     break;
        case SWITCH_ENTER:                  view.OnEnterClick();      break;
        case SWITCH_PAYOUT:                 /* 退币操作 */            break;
        case SWITCH_LEFT:                   view.ChangeHelpPanelActive(); break;
    }
}
```

## 四、硬件灯光控制 (SandboxController)

```csharp
// 所有灯光操作通过 SBoxSandbox 静态方法

// 打开单灯
SBoxSandbox.SwitchOutStateOn(SBOX_SWITCH.SWITCH_A);

// 关闭单灯
SBoxSandbox.SwitchOutStateOff(SBOX_SWITCH.SWITCH_A);

// 批量操作
SandboxController.Instance.TurnOnBetLight();    // 打开A/B/C/D/E 五灯
SandboxController.Instance.TurnOffBetLight();   // 关闭A/B/C/D/E 五灯

SandboxController.Instance.TurnOffTreasureLight(); // 关闭A/B/C 三灯

// 闪烁灯光（中奖时）
SandboxController.Instance.LightFlashing(result, isWin);
// result: 1-5 对应 A-E 按键灯闪烁
// isWin: 是否中奖（控制退币灯闪烁）
```

## 五、硬件状态数据

### 玩家输入状态 (SBoxBattlePlayerInState) — 上行

```csharp
class SBoxBattlePlayerInState
{
    int PlayerId;           // 玩家ID (1~10)
    int LIndex;             // 循环索引 (0~255)
    int state;              // 按键状态 (bit位)
    int LNumberCoinIn;      // 投币计数器 (0~255循环)
    int LNumberCoinOut;     // 退币计数器 (0~255循环)
}
```

state 的 bit 位含义:
```
bit 0:  KEY_A (黑桃)
bit 1:  KEY_B (红心)
bit 2:  KEY_C (梅花)
bit 3:  KEY_D (方块)
bit 4:  KEY_E (皇冠)
bit 5:  KEY_PAYOUT (退币)
bit 6:  KEY_SCORE_UP (上分)
bit 7:  KEY_SCORE_DOWN (下分)
bit 8:  KEY_ROOT (根设置)
bit 9:  KEY_SWITCH (切换)
bit 10: KEY_ESC (取消)
bit 11: KEY_SET (设置)
bit 14: COIN_OUT_TIMEOUT (退币超时)
bit 15: STATE_EXIST (在线标记)
```

### 玩家输出状态 (SBoxBattlePlayerOutState) — 下行

```csharp
class SBoxBattlePlayerOutState
{
    int PlayerId;     // 玩家ID
```
