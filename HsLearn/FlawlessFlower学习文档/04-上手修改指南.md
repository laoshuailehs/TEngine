# FlawlessFlower 上手修改指南

---

## 一、环境搭建

### 必需软件
```
Unity Hub → Unity 2020.3.x (或项目实际版本)
Visual Studio 2022 / Rider
Git (用于版本管理)
```

### 首次打开项目步骤

```bash
# 1. 克隆仓库
git clone <仓库地址>

# 2. 在 Unity Hub 中 "添加项目" → 选择项目根目录

# 3. 等待 Unity 导入资源 (首次可能耗时较长)

# 4. 打开场景
#    Assets/Scenes/ 下找到主场景，双击打开
```

### 运行调试

```
1. Unity Editor 中点击 Play 按钮
2. 编辑器模式下自动创建 BetTestController (F7 开启测试)
3. 使用键盘 F1-F12 和 1-5 数字键进行测试
   (详见 BetTestController.cs 注释)
```

```csharp

## 五、代码规范和约定

### 命名规范
```
类名: PascalCase     → PlayerMgr, SBoxIdea
方法: PascalCase     → GetState(), OnReward()
字段: camelCase      → curGame, isInit
私有: _camelCase 或 camelCase
常量: ALL_CAPS       → SBOX_SWITCH, STATE_IN_KEY_A
```

### 注释风格
```
// ※ 重要逻辑注释
// 普通注释

## 六、常见编译错误解决

| 错误 | 原因 | 解决 |
|------|------|------|
| `CS0246: 找不到类型或命名空间` | 缺少 using 或引用 | 检查是否在正确命名空间 |
| `CS1061: 不包含定义` | 方法签名不对 | 检查参数类型和数量 |
| `NullReferenceException` | 对象未初始化 | 检查 Instance 是否在 Init() 中创建 |
| `MissingComponentException` | 场景缺少组件 | 检查 GameObject 是否挂载脚本 |
| HybridCLR 相关错误 | 热更 DLL 不匹配 | 重新编译替换 DLL |
