## 1. 项目总览

### 1.1 项目简介

这是一个基于 **Unity + SBox 框架** 的博彩类游戏机台项目。SBox 是由黄文（Huang Wen）开发的专用于游戏机台（街机/博彩机）的中间件SDK，提供了**算法卡（IDEA）通信、底板（Sandbox）硬件控制、纸币器管理、打印机控制**等完整的硬件抽象层。

该项目实现了：
- **单机模式**：算法卡运算 + 底板硬件交互
- **局域网联机模式**：通过 WebSocket 实现多机台联网（主从架构）
- **云端联网模式**：通过 HTTP + MQTT 连接后台服务器，支持在线充值和账户管理

### 1.2 核心技术栈

| 层次 | 技术 | 用途 |
|------|------|------|
| 游戏引擎 | Unity 2021+ | 渲染、UI、游戏逻辑 |
| 热更新 | HybridCLR | C# 代码热更新，无需重新打包APK |
| 硬件抽象 | SBox SDK | 算法卡/底板/钱箱/打印机/退币器通信 |
| 局域网通信 | WebSocket (自实现) + UDP | 机台发现与数据交换 |
| 云端通信 | MQTT (MQTTnet) + HTTP | 后台服务器数据交互 |
| 序列化 | Newtonsoft.Json | JSON 序列化/反序列化 |
| 加密 | AES-256-CBC | 通信数据加密 |
| 动画 | DOTween | UI 和游戏动画 |
| 打印 | SkiaSharp + CryPrinter | 小票打印和条码生成 |
| 字体 | BMFont | 位图字体渲染 |
| 日志 | Unity-Logs-Viewer (Reporter) | 游戏运行时日志 |

### 3.1 SBox 框架概述

SBox 是由 **黄文（Huang Wen）** 开发的专用于**游戏机台（街机/博彩机/推币机）**的中介软件 SDK。它提供了从 Unity 到 Android 底层硬件的完整通信桥梁。

### 3.3 SBox 核心数据包结构

```csharp
// 核心数据包 - 所有硬件通信的基础
public class SBoxPacket
{
    public int cmd;      // 命令字 (如 20000=算法复位, 40000=底板复位)
    public int source;   // 源设备地址 (1=App, 2=Idea算法卡, 4=Sandbox底板)
    public int target;   // 目标设备地址
    public int[] data;   // 数据载荷 (int数组, 每4字节一组)
}
```

### 3.4 SBox 设备地址枚举

```

SBOX_DEVICE_SANDBOX = 4   # 底板   (IO控制、钱箱、打印等)
SBOX_DEVICE_CONF    = 8   # 配置存储
SBOX_DEVICE_AUTH    = 16  # 授权验证
SBOX_DEVICE_OTHER   = 32  # 其他设备
```

### 3.5 SBox 主循环流程

```csharp
// SBox.cs 中的 Update() 方法 - 每帧执行
void Update()
{
    if (bInit)
    {
        // 1. 处理 IO 流
        SBoxIOStream.Exec();

        // 2. 处理底板事件 (开关量状态变化检测)
        SBoxSandbox.Exec(millisecond);

        // 3. 处理算法卡事件
        SBoxIdea.Exec(millisecond);

        // 4. 消费 IO 事件队列 (最多50条/帧)
        while (counter++ < 50)
        {
            SBoxPacket packet = SBoxIOStream.Read();
            if (packet != null)
                SBoxIOEvent.SendEvent(packet.cmd, packet); // 按 cmd 分发回调
            else
                break;
        }
    }
}
```

### 3.6 SBox 请求-响应模式

每个 SBox API 都遵循统一的 **请求-响应** 模式：

```csharp
// 以读取算法配置为例
public static void ReadConf()
{
    // 1. 构造请求数据包
    SBoxPacket sBoxPacket = new SBoxPacket(cmd: 20002, source: 1, target: 2, size: 2);
    sBoxPacket.data[0] = 0;
    sBoxPacket.data[1] = 0;

    // 2. 注册回调监听 (以 cmd 为 key)
    SBoxIOEvent.AddListener(sBoxPacket.cmd, ReadConfR);

    // 3. 发送请求
    SBoxIOStream.Write(sBoxPacket);
}

// 4. 响应回调 (由主循环 SBoxIOEvent.SendEvent 触发)
private static void ReadConfR(SBoxPacket sBoxPacket)
{
    SBoxConfData conf = new SBoxConfData();
    conf.result      = sBoxPacket.data[0];
    conf.MachineId   = sBoxPacket.data[6];
    conf.CoinValue   = sBoxPacket.data[11];
    // ... 解析所有配置字段

    // 5. 通过 EventCenter 通知上层
    EventCenter.Instance.EventTrigger(SBoxEventHandle.SBOX_READ_CONF, conf);
}
```

## 4. 网络模块详解

本项目网络架构分为 **三个层次**：

| 层次 | 协议 | 用途 | 关键类 |
|------|------|------|--------|
| 云端通信层 | HTTP + MQTT | 后台服务器数据交换 | `Net.cs` |
| 局域网通信层 | WebSocket + UDP | 机台间主从通信 | `NetMgr.cs`, `ServerWS.cs`, `ClientWS.cs` |
| WebSocket 底层 | TCP (自实现) | WebSocket 协议实现 | `Frame.cs`, `ClientConnection.cs`, `WebSocketServer.cs` |

```csharp
public class ClientConnection
{
    TcpClient m_tcpClient;
    public event Action<ClientConnection, string> ReceivedTextualData;
    public event Action<ClientConnection> Disconnected;

    // 开始异步接收数据
    public void StartReceiving()
    {
        SocketAsyncEventArgs args = new SocketAsyncEventArgs();
        args.Completed += OnDataReceived;
        m_tcpClient.Client.ReceiveAsync(args);
    }

    // 数据接收回调
    void OnDataReceived(object sender, SocketAsyncEventArgs e)
    {
        if (HandleFrame(e))
            StartReceiving();  // 继续监听
    }

    // 处理 WebSocket 帧
    bool HandleFrame(SocketAsyncEventArgs args)
    {
        Frame frame = Frame.FromBuffer(args.Buffer);

        if (frame.Opcode == Frame.Opcodes.Close)
        {
            Disconnect();
            return false;
        }

        if (frame.Opcode == Frame.Opcodes.Text)
            ReceivedTextualData(this, Encoding.UTF8.GetString(frame.UnmaskedPayload));

        return true;
    }
}
```

```csharp
        AddEventListener();

        // 4. 等待硬件就绪
        StartCoroutine(CheckSBoxReady());
    }

    IEnumerator CheckSBoxReady()
    {
        while (!SBoxIdea.Ready())
            yield return new WaitForSeconds(0.5f);

        // 算法卡就绪，执行复位
        SBoxIdea.Reset();
    }
}
```

```csharp
public class LoginManager : MonoBehaviour
{
    [SerializeField] string serverUrl = "https://your-server.com/machine";
    string baseKey = "your-base64-aes-key";

    public IEnumerator Login(string machineCode)
    {
        // 1. 加密机器码
        string json = $"{{\"machineCode\":\"{machineCode}\",\"lang\":\"en\"}}";
        byte[] iv = AESScript.GenerateIv();
        string encrypted = AESScript.Encrypt(json, baseKey, iv);

        // 2. 发送 POST 请求
        WWWForm form = new WWWForm();
        form.AddField("ctxt", encrypted);
        form.AddField("lang", "en");

        UnityWebRequest request = UnityWebRequest.Post($"{serverUrl}/login", form);
        yield return request.SendWebRequest();

        if (request.result == UnityWebRequest.Result.Success)
        {
            // 3. 解析响应
            PostMesg mesg = JsonConvert.DeserializeObject<PostMesg>(
                request.downloadHandler.text);

            if (mesg.Code == 0)
            {
                // 4. 解密动态密钥
                string decrypted = AESScript.Decrypt(mesg.Data.ctxt, baseKey);
                PostData data = JsonConvert.DeserializeObject<PostData>(decrypted);

                // 5. 连接 MQTT
                yield return ConnectMQTT(data.mqttBroker, data.accessToken);
            }
        }
    }

    async Task ConnectMQTT(string brokerUrl, string token)
    {
        var factory = new MqttFactory();
        var client = factory.CreateMqttClient();

        var options = new MqttClientOptionsBuilder()
            .WithWebSocketServer(o => o.WithUri(brokerUrl))
            .Build();

        await client.ConnectAsync(options);

        // 发布登录 Token
        var msg = new MqttApplicationMessageBuilder()
            .WithTopic("machine/HD_login/1")
            .WithPayload(JsonConvert.SerializeObject(new { token }))
            .Build();

        await client.PublishAsync(msg);

        // 订阅响应
        await client.SubscribeAsync("machine/HD_login/1");
    }
}
```

```csharp
public static class CryptoHelper
{
    static string defaultKey = "your-fixed-base64-key";
    static string dynamicKey = null;

    public static string EncryptForSend(string json)
    {
        string key = dynamicKey ?? defaultKey;
        byte[] iv = GenerateRandomIV();
        using (Aes aes = Aes.Create())
        {
            aes.Key = Convert.FromBase64String(key);
            aes.IV = iv;
            aes.Mode = CipherMode.CBC;
            aes.Padding = PaddingMode.PKCS7;

            using (var encryptor = aes.CreateEncryptor())
            {
                byte[] plainBytes = Encoding.UTF8.GetBytes(json);
                byte[] cipherBytes = encryptor.TransformFinalBlock(plainBytes, 0, plainBytes.Length);

                // 将 IV 拼接在密文前面
                byte[] result = new byte[iv.Length + cipherBytes.Length];
                Buffer.BlockCopy(iv, 0, result, 0, iv.Length);
                Buffer.BlockCopy(cipherBytes, 0, result, iv.Length, cipherBytes.Length);

                return Convert.ToBase64String(result);
            }
        }
    }

    public static string DecryptReceived(string cipherText)
    {
        string key = dynamicKey ?? defaultKey;
        byte[] fullData = Convert.FromBase64String(cipherText);

        // 提取 IV (前16字节)
        byte[] iv = new byte[16];
        Buffer.BlockCopy(fullData, 0, iv, 0, 16);

        // 提取密文
        byte[] cipherBytes = new byte[fullData.Length - 16];
        Buffer.BlockCopy(fullData, 16, cipherBytes, 0, cipherBytes.Length);

        using (Aes aes = Aes.Create())
        {
            aes.Key = Convert.FromBase64String(key);
            aes.IV = iv;
            aes.Mode = CipherMode.CBC;
            aes.Padding = PaddingMode.PKCS7;

            using (var decryptor = aes.CreateDecryptor())
            {
                byte[] plainBytes = decryptor.TransformFinalBlock(cipherBytes, 0, cipherBytes.Length);
                return Encoding.UTF8.GetString(plainBytes);
            }
        }
    }
}
```

### B. 安全注意事项

1. **密钥管理**: PostBase64Key 硬编码在代码中，生产环境建议使用更安全的密钥存储方案
2. **HTTPS 证书**: 项目中使用了绕过证书验证的 `BypassCertificate`，生产环境应使用正确的证书验证
3. **密码安全**: 管理后台密码应使用更安全的哈希算法，而非明文传输
4. **日志安全**: Reporter 日志组件可能泄露敏感信息，发布版本应关闭

### C. 性能优化建议

1. **对象池**: 频繁创建的 SBoxPacket 可以使用对象池减少 GC
2. **异步 IO**: MQTT 和 WebSocket 已使用异步模式，注意不要阻塞主线程
3. **资源加载**: ResMgr1001 已支持 AssetBundle，可进一步实现异步加载
4. **UI 优化**: SuperScrollView 已使用虚拟列表，大数据量场景性能良好
